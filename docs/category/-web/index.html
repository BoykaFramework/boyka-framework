Web

📄️ 🪛 Setup Configuration file

Before starting to automate Web applications, we need to first setup the configuration file for our Application under test.

📄️ 📄 Create Page Object

In Boyka, we have decoupled the page object from the WebDriver class. This is done by creating a Page Object class and defining locators using Locator builder class.

📄️ ✅ Write Test for Web

Once the page object class is created, we can use it to interact with the page in our tests by passing the locators to static action methods exposed by Boyka framework.
