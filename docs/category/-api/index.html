API

📄️ 🪛 Setup Configuration file

We can set multiple configurations in the configuration file with different key name for different end points.

📄️ 🏗️ Compose Request

For API automation, we have created ApiRequest class which you can utilize to build API requests.

📄️ 🔫 Execute Request

Once you have composed your API request, you can execute it using class ApiManager and method execute.

📄️ ✅ Verify Response

Once the ApiResponse object is returned from the execute method, it can be used to verify that the response is valid.
